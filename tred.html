TRED

a sneaker resale concept, coded from scratch.

Project Overview

Tred is a concept homepage I designed and developed as my final exam project during my Web Design diploma in 2023. It’s a sneaker reseller platform that features top sneaker brands, along with merch, apparel, and accessories — all presented in a clean, urban-inspired layout.

Role

Web Designer / Dev

Duration

30 days

The goal was to create a modern, responsive landing page that highlights featured drops and exclusive releases. I built it using HTML, CSS, JavaScript, and connected it to a local server with PHP and phpMyAdmin. The site runs on localhost and is fully responsive across devices.
